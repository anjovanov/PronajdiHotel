# PronajdiHotel

A web app that allows you to Add/Remove/Edit and Browse through hotels, and leave 1-5 star reviews.

Technologies used:

- Javascript
- NodeJS
- ExpressJS
- MongoDB
- Bootstrap

All other dependencies can be found in the package.json file.
